# api/data/

Data files for seeding the Pikly database. JSONL files are excluded from git.

## Required

Place your data file here before running the seeder:

```
api/data/products_cleaned.jsonl
```

Optionally run the Discovery Engine first for AI recommendations:

```bash
cd pipeline
python hybrid_discovery_engine.py \
  --input ../api/data/products_cleaned.jsonl \
  --output ../api/data/products_discovery_enhanced.jsonl
```

The seeder auto-prefers `products_discovery_enhanced.jsonl` over `products_cleaned.jsonl`.

## Seed Commands

```bash
cd api
npx ts-node scripts/seed-categories-pg.ts   # categories first
npx ts-node scripts/seed-pg.ts              # all products
npx ts-node scripts/sync-algolia-pg.ts      # push to Algolia
```

## Files

| File | Status |
|---|---|
| `products_cleaned.jsonl` | Required — not in git (add manually) |
| `products_discovery_enhanced.jsonl` | Optional — generated by discovery engine |
| `banners.json` | Seed data for homepage banners |
| `coupons.json` | Seed data for discount coupons |
| `categories.json` | Legacy — not used by current seeder |
| `users.json` | Legacy — not used by current seeder |
| `orders.json` | Legacy — not used by current seeder |
